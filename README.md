# SRCNN-PyTorch

Lightweight PyTorch implementation of SRCNN (image super-resolution) with training, evaluation, and plotting utilities.

## Overview
- Implements a 3-layer SRCNN (9-5-5 kernels) for single-channel (Y) image super-resolution.
- Provides HDF5-backed datasets, training loop with checkpointing, inference script, and experiment visualization (PSNR plots).

## Quickstart
1. Install dependencies (recommended Python 3.8+):

```bash
pip install torch torchvision numpy pillow h5py tqdm pandas seaborn matplotlib tabulate
```

2. Train (example):

```bash
python x4/train.py --train-file "91-image_x3.h5" \
    --eval-file "Set5_x3.h5" --outputs-dir "outputs/x3_lr1e-4" \
    --scale 3 --lr 1e-4 --batch-size 16 --num-epochs 100 --num-workers 2 --seed 123
```

3. Run inference on an image:

```bash
python x4/test.py --weights-file "outputs/x3_lr1e-4/x3/best.pth" \
    --image-file "data/butterfly_GT.bmp" --scale 3
```

4. Plot training results and summary table:

```bash
python x4/plot_results.py
python x4/summary_table.py
```

5. Optional: open the demo notebook: `x4/ML_Project.ipynb`

## Folder layout
- `x4/` — model, training, test, dataset and utility scripts
- `data/` — input images and HDF5 datasets (not tracked in repo)
- `outputs/` — checkpoints and logs generated by training

## Recommended repo name
`SRCNN-PyTorch` — concise and discoverable for PyTorch SRCNN experiments.

## Notes
- Large data files and model checkpoints are ignored via `.gitignore`.
- Modify hyperparameters and paths as needed for your experiments.
